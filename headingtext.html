Lesson 3 - Heading for text




We will learn to create text using the heading tag. Here is how to do it. Do a tag, then put in h1 or h2 or so on, then close the tag. H1 is the biggest type of heading, then h2 is the second smallest, then h3 is the third smallest, and so on. It goes on till about h6. The only way to find out is to code! Try it yourself
